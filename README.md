# 🎯 Quiz Game Application

An interactive and responsive **Quiz Game Application** built using **HTML, CSS, and JavaScript** as part of the **SkillCraft Technology Web Development Internship – Task 3**.

The application presents multiple-choice questions, validates user answers, tracks the score, and displays the final result after the quiz is completed.

---

## 🚀 Features

- 📝 15 Multiple Choice Questions
- ✅ Instant Answer Validation
- 🎨 Correct & Incorrect Answer Highlighting
- 📊 Score Calculation
- 🔄 Restart Quiz Functionality
- 📱 Fully Responsive Design
- 💻 Clean and Modern User Interface
- ⚡ Built with Vanilla JavaScript

---

## 🎮 How to Play

1. Read the question carefully.
2. Select one answer from the available options.
3. The selected answer will be highlighted:
   - 🟢 Green for the correct answer
   - 🔴 Red for an incorrect answer
4. Click **Next Question** to continue.
5. After completing all questions, your final score will be displayed.
6. Click **Play Again** to restart the quiz.
